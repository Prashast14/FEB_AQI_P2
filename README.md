# AirPure Innovations - Air Quality Index Analysis Project

## Project Overview
This project analyzes Air Quality Index (AQI) data across Indian cities to provide insights and visualizations through Power BI dashboards.

## Project Structure
```
FEB_AQI_P2/
├── data/                    # Data files and datasets
│   ├── raw/                 # Original, unprocessed data
│   ├── processed/           # Cleaned and transformed data
│   └── exports/             # Exported results and reports
├── database/                # Database scripts and schemas
│   ├── schema/              # Table definitions and DDL scripts
│   ├── queries/             # SQL queries and views
│   └── backups/             # Database backups
├── powerbi/                 # Power BI related files
│   ├── dashboards/          # .pbix dashboard files
│   ├── templates/           # Dashboard templates
│   └── assets/              # Images and other assets
├── scripts/                 # Analysis and utility scripts
│   ├── python/              # Python scripts for data processing
│   ├── sql/                 # SQL scripts
│   └── automation/          # Automation scripts
├── docs/                    # Documentation
│   ├── requirements/        # Project requirements
│   ├── design/              # Design documents
│   └── reports/             # Analysis reports
└── config/                  # Configuration files

```

## Getting Started
1. Clone this repository
2. Review the documentation in the `docs/` folder
3. Set up the database using scripts in `database/schema/`
4. Open Power BI dashboards from `powerbi/dashboards/`
